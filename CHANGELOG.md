# Changelog

Todas las versiones y cambios de este proyecto se documentarán en este archivo.

## [1.0.0] - 2026-04-15
### Añadido
- Motor de segmentación basado en OpenCV para extracción de componentes.
- API RESTful en FastAPI.
- Integración de Inpainting (Telea algorithm) para reconstrucción dinámica de fondo.
- Motor de renderizado Frontend basado en Fabric.js.
- Editor de capas visual con soporte jerárquico.
- Extracción e interpretación de texto mediante OCR (pytesseract).
- Script de despliegue automático hacia GitHub.
